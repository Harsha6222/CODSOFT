Tic-Tac-Toe Game (Console-based)
This is a simple console-based Tic-Tac-Toe game developed in C++ as part of the CodSoft Virtual Internship. The game allows two players to play the classic game directly through the terminal or command prompt. It includes turn-based gameplay, win condition checks, and input validation.

Features
Two-player gameplay with players taking turns to place X and O.
Dynamic board display that updates after every move.
Input validation to ensure players enter valid positions.
Win detection to check for winning combinations.
Option to restart the game once it ends.

How to Play
Player 1 (X) and Player 2 (O) take turns to enter a number (1-9) corresponding to a cell on the board.
The board will be displayed after each move.
The game ends when one player wins, or if the board is full resulting in a draw.
